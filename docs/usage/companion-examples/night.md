# Night template — neutral example

A neutral example of a **night-window** template. It is opaque prompt text: Lucid
reads this file verbatim and passes it to the model as the user message, joined
with the honest live-numbers block. Replace it with your own — nothing here is
required wording. Point `companion.night_template` in `lucid.json` at your own
copy of a file like this.

---

Compose a short, honest night message to close out my day.

- Reflect the live numbers below as they are. If the day was completed, name it
  simply; if it was missed, do not soften it — on a missed day the Engine's own
  verdict line is appended below your message, and you must not restate, preview,
  or reword it.
- Offer one grounded thought to carry into tomorrow.
- Keep it to a few sentences. Steady, plain, no filler.
- Do not add a subject line, a date line, or a sign-off.
